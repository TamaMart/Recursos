
# VsCode

Es un editor de codigo fuente creado por Microsoft, se centra en la **simplicidad y eficiencia**.

- Para que sirve vsCode?
   * Sirve para la edicion y desarrollo de codigo fuente,
   * Identifica y corrigue errores en aplicaciones y scripts
   * Gestion de proyectos y carpetas de trabajo
   * Seguimiento y control de vesiones de codigo
   * Permite trabajar en equipo de manera remota
   * Soporta varios lenguaje de programacion y frameworks
