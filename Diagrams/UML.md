# UML

- ¿Qué es uml y para qué sirve? 

 UML significa Unified Modeling Language (Lenguaje de Modelado Unificado). Es un lenguaje visual estándar que se usa para diseñar y documentar sistemas de software antes o durante su desarrollo.
Sirve para:
1. Planificar antes de programar:
Permite diseñar la estructura del sistema sin escribir código todavía, detectando problemas antes de que sean costosos.
2. Comunicar ideas:
Un diagrama UML lo entiende cualquier programador del mundo, sin importar el lenguaje que use (Python, Java, C++, etc.).
3. Documentar el sistema:
Sirve como manual visual de cómo está construido el software.
4. Trabajar en equipo:
Permite que varios desarrolladores trabajen sobre el mismo sistema sin confusiones.
