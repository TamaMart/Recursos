# Operaciones Basicas

- Entrero es un número sin decimales, se acorta como **INT** En este caso, el número es 1
- Decimal es un número con decimales, se acorta como **FLOAT** En este caso, el número es 1.5.
- Booleano es un valor que puede ser verdadero o falso, se acorta como **BOOL** En este caso, el valor es True.

Variable
`num_1 = 5`
`num_2 = 3`

* Suma
 `print("suma:", num_1 + num_2)`

* Resta
 `print("resta:", num_1 - num_2)`

* Multiplicacion
`print("multiplicación:", num_1 * num_2)`

* División
`print("división:", num_1 / num_2)`

* Modulo
`print("módulo:", num_1 % num_2)`

* Exponente
`print("exponente:", num_1 ** num_2)`

* Mayor que
`print("mayor que",1 > 2)`

* Menor que
`print("menor que",1 < 2)`

* Igual a
`print("igual a",1 == 2)`

* Diferente de
`print("diferente a",1 != 2)`

* Mayor o igual 
`print("mayor o igual que",1 >= 2)`   

Promedio

`media = (num_1 + num_2) / 2`

`print("Nota:", media)`

`print("Aprobado:", media >= 4)`
